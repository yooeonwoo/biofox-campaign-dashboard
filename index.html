시딩 전략

접기 ▲
🚫
인셀덤 느낌 절대 금지 — 나이 든 원장님들의 저품질 대량 콘텐츠 = MLM 이미지. 이전 접근법(원장님 대량 콘텐츠)은 폐기.

전략 전환: 양 → 질

🚫 이전 (폐기)
  • 원장님 대량 콘텐츠 발행
  • 원장님 중심 주체
  • 수량 KPI 중심
  • 전문가 콘텐츠 위주
→
✅ 현재 (확정)
  • 고품질 소비자 지향 콘텐츠
  • 마이크로 인플루언서 (20~30대)
  • 퀄리티 게이트 (기준 미달 시 미발행)
  • 소비자가 저장/북마크하고 싶은 콘텐츠

마이크로 인플루언서

👤 타겟 프로필
  • 연령: 20~30대
  • 젊은 사람이 쓰는 것 → 나이 든 사람이 따라하는 구조
  • 뷰티과 학생: 좋은 잠재력, 코칭 세션 필요
📸 콘텐츠 기준
  • 소비자가 저장/북마크하고 싶은 퀄리티
  • Before/After: 드라마틱하게 (메이크업+스타일링 포함)
  • 민낯 비교 X
  • 무상 제품 제공 = 퀄리티 기준 설정 가능

콘텐츠 레퍼런스 템플릿 (3종)

B2B vs B2C 분리

접기 ▲
B2B 원장님 타겟
🏥
  • 전문가 콘텐츠 (시술 장면, 데이터)
  • 상담 무기 제공
  • 재방문 구조 설계
  • 전문성 강조: 진짜 피부 문제 해결
  • 원장님 얼굴 노출 불필요
B2C 소비자 타겟
💁‍♀️
  • 마이크로 인플루언서 콘텐츠
  • 소비자 시딩 (저장/북마크 유도)
  • 리얼 애프터 케어 제품 구매 연결
  • 예쁘고 트렌디한 느낌
  • 바이럴 가능한 콘텐츠
🏥

4P 전략

접기 ▲
P
📦 Product
B2B: BIOFOX 천방케어 시술 솔루션
B2C: 리얼 애프터 케어 (미스트 + 재생크림) — 시술 후 24시간 홈케어
P
💰 Price
프리미엄 포지셔닝

📍 Place
인스타(메인) / 유튜브(검색 공백 선점) / 블로그(SEO)
B2B: 원장님 직접 채널 + 챌린지
B2C: 마이크로 인플루언서 + 송선미님 채널
P
📣 Promotion
6~8월 시딩 → 9월 광고 집행 + PPL
송선미님 자연스러운 노출 → 검색 유도 → 고품질 콘텐츠 발견

Do / Don't

접기 ▲
✅
✅ DO — 반드시 지키기
  • ✅ 고품질 소비자 지향 콘텐츠만 발행
  • ✅ 드라마틱 Before/After (메이크업+스타일링)
  • ✅ 20~30대 마이크로 인플루언서 활용
  • ✅ 천방케어를 항상 전면에
  • ✅ 검색 결과 상위 15개 품질 관리
  • ✅ "관리하는 여자들은 다 저거 하네" 느낌
  • ✅ 샘플 먼저 → 피드백 → 확산
  • ✅ B2B/B2C 콘텐츠 명확 분리
  • ✅ 자연스러운 변화 스토리
🚫
🚫 DON'T — 절대 금지
  • ❌ 인셀덤 느낌 (저품질 대량 콘텐츠)
  • ❌ 나이 든 원장님 얼굴 전면 노출
  • ❌ 청담엑스날 단독 전면 노출
  • ❌ 레아쥬 단독 전면 노출
  • ❌ 711 MEDISHOT 혼합
  • ❌ 기준 미달 콘텐츠 발행
  • ❌ 민낯 Before/After (드라마틱하지 않음)
  • ❌ 수량 KPI 중심 운영
  • ❌ 성형 느낌의 메시지
